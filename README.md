# Automated File Organizer

## Description
A Python application that automatically organizes files into folders based on their file types.

## Features
- Organizes Images
- Organizes Documents
- Organizes Audio Files
- Organizes Video Files
- Automatically creates folders

## Technologies Used
- Python
- OS Module
- Shutil Module

## How to Run

1. Install Python
2. Run:
   python file_organizer.py
3. Enter the folder path you want to organize
